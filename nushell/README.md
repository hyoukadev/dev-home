# Install

## Windows
```sh
winget install nushell
winget install JanDeDobbeleer.OhMyPosh -s winget
```

## MacOS

```sh
cargo install nu --locked
brew install jandedobbeleer/oh-my-posh/oh-my-posh
```


## Ubuntu

```sh
cargo install nu --locked
curl -s https://ohmyposh.dev/install.sh | bash -s
oh-my-posh init nu
```
